# Tour Detail Screen

## Overview

Detailed view of a specific tour including description, itinerary, pricing, schedules, reviews, and booking functionality.

## Design Reference

- **Design file**: `docs/Fe/design/tour_booking_details/screen.png`
- **HTML reference**: `docs/Fe/design/tour_booking_details/code.html`

## Screen Information

| Item | Value |
|------|-------|
| Screen ID | `SCR-004` |
| Route | `/tours/[slug]` |
| Access | Public (booking requires auth) |
| Layout | Main Layout |

## Key Features

1. Image gallery with hero
2. Experience highlights (icons)
3. Tour description and itinerary
4. Schedule picker with availability
5. Traveler selection (Adult/Child)
6. Price calculation
7. Reviews section
8. Map with meeting point
