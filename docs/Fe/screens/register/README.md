# Registration Screen

## Overview

The registration screen allows new users to create an account with email/password or social providers.

## Design Reference

- **Design file**: `docs/Fe/design/user_registration_page/screen.png`
- **HTML reference**: `docs/Fe/design/user_registration_page/code.html`

## Screen Information

| Item | Value |
|------|-------|
| Screen ID | `SCR-002` |
| Route | `/register` |
| Access | Anonymous only |
| Layout | Auth Layout |

## Key Features

1. Registration form with validation
2. Password strength indicator
3. Country/phone selection
4. Terms of Use agreement
5. Social registration (Google, Facebook)
6. Hero image with quote
