# Admin Dashboard Screen

## Overview

The admin dashboard provides administrators with a comprehensive overview of the booking platform including revenue, bookings, tours, and user statistics.

## Design Reference

- **Design file**: `docs/Fe/design/admin_dashboard/screen.png`
- **HTML reference**: `docs/Fe/design/admin_dashboard/code.html`

## Screen Information

| Item | Value |
|------|-------|
| Screen ID | `SCR-010` |
| Route | `/admin/dashboard` |
| Access | ADMIN only |
| Layout | Admin Layout (sidebar) |

## Key Features

1. Statistics cards (Revenue, Bookings, Tours, Users)
2. Revenue trend chart
3. Quick action buttons
4. Recent bookings table
5. Top performing tours list
