# Tours Overview Screen (Home Page)

## Overview

The main landing page displaying available tours with search, filtering, and pagination features.

## Design Reference

- **Design file**: `docs/Fe/design/current_tours_overview/screen.png`
- **HTML reference**: `docs/Fe/design/current_tours_overview/code.html`

## Screen Information

| Item | Value |
|------|-------|
| Screen ID | `SCR-003` |
| Route | `/` or `/tours` |
| Access | Public |
| Layout | Main Layout |

## Key Features

1. Hero section with search
2. Tour cards grid with images
3. Price display and ratings
4. Filter by category (Popular, Price, Difficulty)
5. Pagination
6. Footer with company info
